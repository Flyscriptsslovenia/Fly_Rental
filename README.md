FLY SCRIPTS - Rent A Car

## HOW TO MAKE SCRIPT WORK

1. Drag and drop folder 'fly_rental' in your resources folder
2. Type 'start fly_rental' in server.cfg
3. Restart your server
4. Test it
5. Enjoy
